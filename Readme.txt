Relevant code for the following publication:

Rahul Jha, Catherine Finegan-Dollak, Ben King, Reed Coke and Dragomir Radev. Content Models for Survey Generation: A Factoid-Based Evaluation. Annual Meeting of the Association for Computational Linguistics, 2015.

-----

Python implementation for TopicSum and HitSum.

TopicSum implementation is based on:

http://www.mblondel.org/journal/2010/08/21/latent-dirichlet-allocation-in-python/
https://github.com/rebeccamason/codesample/tree/master/TopicSum

HitSum code will need to be adapted to suit your needs, I have added comments wherever possible.
